> 수학은 사고의 뼈대다. 차별점은 여기서 나온다.

## 회차

| 회차 | 문제 상황 | A층 |
|---|---|---|
| W1 | 변수가 많다 | ALS, Matrix Factorization, SVD |
| W2 | 뭘 해야 할지 모른다 | Gradient Descent, Backpropagation, MLP |
| W3 | 데이터로 답을 찾는다 | MLE, MAP, Bayesian, 정규화 |
| W4 | 전부 볼 수 없다 | Sampling, 가설 검정, Multi-Armed Bandit |
| W5 | 정확함이 비싸다 | Bloom, HyperLogLog, Count-Min, t-digest |
| W6 | 하나로는 부족하다 | Boosting, XGBoost, Random Forest |
| W7 | 개체가 아니라 연결이 중요하다 | PageRank, Power Iteration |
| W8 | 언제 멈출 것인가 | Optimal Stopping, Secretary Problem |
| W9 | 시스템이 왜 느린지 모른다 | Little's Law, Queueing |
