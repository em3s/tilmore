## 회차 구성

각 회차는 두 층:

- **A층 — 통찰** — 핵심 통찰과 수학적 형식화. 본편.
- **B층 — 도구** — 같은 문제 계열의 관련 알고리즘. 보너스.

## 각 회차의 섹션

- **한눈에** — 회차 흐름을 한 화면에 잡는 미리보기. 5개 안팎의 짧은 bullet
- **문제 상황** — 엔지니어가 실무에서 마주치는 풍경. 추상 → 구체 예시
- **핵심 통찰** — 그 상황에서 어떻게 사고하는가. 인용 한 줄 + 단계
- **한 사례 — [도메인]** — 통찰의 출처. 회차가 빌려온 수학 사례
- **수학/도출 섹션들** — 본격 식 전개
- **변주 섹션들** — 같은 통찰의 다른 얼굴 (임베딩, SVD, 암묵적 피드백 등)
- **직접 돌려보기** — 데이터셋 정의 + 실행 가능한 최소 코드
- **마무리** — 통찰 회귀 + 실무 4갈래 + 현장 인용
- **부록**
  - **건드리기만 하는 것** — 이름과 존재만 언급
  - **안 다루는 것** — 명시적으로 제외. 범위 밖
  - **보너스 도구 (B층)** — 추가로 다룰 수 있는 관련 기법
  - **핵심 키워드** — 검색 질의어. 영문 우선

## 용어 표기

**한국어(영어) 병기**. 첫 등장 시 한 번. 이후엔 자연스러운 쪽 사용.

## 노테이션

수식이 처음이거나 오래된 분들을 위한 빠른 참조. 본문 전반에 반복되는 것만 모았다. 회차별 특수 기호는 그 회차에서 따로 설명한다.

### 벡터·행렬

| 기호 | 읽는 법 | 의미 |
|---|---|---|
| $\mathbf{v}$ | "벡터 v" | 굵은 소문자는 벡터 (열벡터, 1차원 배열) |
| $A$ | "행렬 A" | 대문자는 행렬 (2차원 배열) |
| $A \in \mathbb{R}^{m \times n}$ | "A는 m 곱하기 n 실수 행렬" | $m$행 $n$열, 원소는 실수 |
| $A^\top$ | "A 트랜스포즈" | 전치 — 행과 열을 뒤집음 |
| $AB$ | "A B" | 행렬 곱 |
| $A_{ij}$, $a_{ij}$ | "A i j 성분" | $A$의 $i$행 $j$열 원소 |
| $\mathbf{u}_i$ | "u 서브 i" | $i$번째 벡터 (문맥에 따라 행렬의 $i$번째 행) |
| $I$ | "항등 행렬" | 대각이 1, 나머지 0 |

### 집합·합·인덱싱

| 기호 | 읽는 법 | 의미 |
|---|---|---|
| $\mathbb{R}, \mathbb{Z}, \mathbb{N}$ | "실수/정수/자연수 집합" | 수의 종류 |
| $i \in S$ | "i는 S의 원소" | $i$가 집합 $S$에 속함 |
| $S \subset T$ | "S는 T의 부분집합" | 포함 |
| $\sum_{i=1}^{n} x_i$ | "i 1부터 n까지 x i의 합" | 합 |
| $\sum_{i \in S} x_i$ | "S에 속한 i에 대한 x i의 합" | 집합 위에서의 합 |
| $A_S$ | "A의 S 인덱싱" | $A$에서 $S$에 해당하는 행/열만 추린 부분 |

### 거리·크기

| 기호 | 읽는 법 | 의미 |
|---|---|---|
| $\Vert \mathbf{v} \Vert$ | "벡터 v의 크기" | 벡터의 길이. 보통 유클리드 거리 |
| $\Vert \mathbf{v} \Vert^2$ | "v의 크기 제곱" | 각 원소를 제곱해서 합 |
| $\Vert A \Vert_F$ | "A의 프로베니우스 크기" | 행렬의 모든 원소를 제곱해서 합한 뒤 제곱근 |
| $\mathbf{u}^\top \mathbf{v}$ | "u 트랜스포즈 v" | 내적 — 같은 자리 원소를 곱한 값들의 합 |

### 최적화

| 기호 | 읽는 법 | 의미 |
|---|---|---|
| $\min_{x} f(x)$ | "x에 대한 f의 최솟값" | $x$를 바꿔가며 얻는 $f$의 최솟값 |
| $\arg\min_{x} f(x)$ | "f를 최소화하는 x" | 최솟값을 만드는 $x$ 자체 |
| $\nabla f$ | "그래디언트 f" | 다변수 미분. 가장 가파르게 증가하는 방향 |
| $\lambda$ | "람다" | 보통 정규화(regularization) 계수 |

### 확률·통계

| 기호 | 읽는 법 | 의미 |
|---|---|---|
| $P(A)$ | "A의 확률" | 사건 $A$가 일어날 확률 |
| $P(A \mid B)$ | "B 주어졌을 때 A의 확률" | 조건부 확률 |
| $X \sim \mathcal{N}(\mu, \sigma^2)$ | "X는 정규분포를 따른다" | 분포를 따른다 |
| $\mathbb{E}[X]$ | "X의 기댓값" | 평균 |
| $\mathrm{Var}(X)$ | "X의 분산" | 분산 |

### 그리스 문자 (자주 등장)

| 기호 | 발음 | 자주 쓰이는 곳 |
|---|---|---|
| $\alpha$ | 알파 | 학습률, 신뢰도 계수 |
| $\beta$ | 베타 | 회귀 계수 |
| $\gamma$ | 감마 | 할인 인자 |
| $\delta$ | 델타 | 변화량, 작은 양 |
| $\epsilon$ | 입실론 | 오차, 매우 작은 양 |
| $\theta$ | 세타 | 일반 파라미터 |
| $\lambda$ | 람다 | 정규화 계수, 고윳값 |
| $\mu$ | 뮤 | 평균 |
| $\sigma, \Sigma$ | 시그마 (소·대문자) | 표준편차 / 공분산 행렬, 합 기호 |
| $\Omega, \omega$ | 오메가 | 관측 집합, 가중치 |
